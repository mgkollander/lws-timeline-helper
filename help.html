LWS Timeline Helper — How To Use ← Back to Timeline

Overview

This timeline tool was created for Last War: Survival to help players fully optimize rewards for Arms Race and VS Alliance Duel. Additionally, users can create custom events and tasks of their own, such as alliance-specific objectives (e.g., Marshal's Guard, final pushes).

Buttons & Drop-Downs

  • Timezone Select: Choose from the dropdown. Use “Sync to Local Time” to switch to your device’s timezone.
  • Auto‑Scroll: When enabled, the timeline stays centered on the current time (updated by minute).
  • Zoom: Use +/− buttons to change how many minutes each cell represents on the timeline.

Timeline Events

  • Curated events: Arms Race and VS Alliance Duel events/tasks are built-in and cannot be changed.
  • Custom events: These are your own custom events, saved locally on your browser for the current week.
  • Event colors: The colored top border of each event can be used to spot double‑dipping opportunities. When a VS Alliance Duel and an Arms Race event share the same color at the same time (e.g., purple), completing the matching Arms Race tasks earns points in both events.

Tasks

  • Task checkboxes: Task completion is saved per week.
  • "What Do I Need To Do Right Now?": This section shows tasks for events currently in progress, ordered by shortest event first.
  • "Upcoming Tasks": This section shows all events starting within the next 6 hours, ordered by shortest event first then by start time.

Add Custom Events

  • Fill in Title, Day (starts at today), Start/End, Color, and optional Tasks (comma-separated).
  • Click Add Custom Event to place it on this week’s timeline.
  • Use Clear All to remove only the current week’s custom events.
  • Delete individual custom events from the list below the form.

Note: Custom events are saved to your browser’s local storage and do not sync across devices. If you clear your browser cache, you will lose your custome events!

Privacy

  • Where your data lives: All custom events and task statuses are stored locally in your browser.
  • External storage: Absolutely nothing is uploaded or shared.
